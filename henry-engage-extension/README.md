# Henry Engage — Chrome Extension

Capture web content, highlights, and notes directly into Henry AI.

## Install (takes 30 seconds)

1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select this `henry-engage-extension/` folder
5. Henry Engage appears in your toolbar

Henry AI must be running on your Mac for the extension to work.

## How to use

**Floating button:** Select any text on a webpage (10+ chars) → a purple "Send to Henry" button appears → click it

**Right-click menu:**
- Select text → right-click → "Send to Henry: ..."
- Right-click page → "Send page summary to Henry"
- Right-click any link → "Send link to Henry"

**Popup (click toolbar icon):**
- 📋 Selected text — grab current selection
- 📄 This page — capture title + URL
- 🔗 Page URL — capture just the URL
- 💬 Quick note — type anything
- Choose category (Note, Quote, Task, Idea, Web clip, Link, Reminder)
- ⌘+Enter to send

## Captured content

Everything lands in Henry's **Captures panel** (⊕ in sidebar). From there you can:
- Route to Tasks, Reminders, Journal, Memory
- Send to Henry for AI processing
- Edit before routing
- Archive when done

## Requirements

- Henry AI desktop app running
- Mac (Henry runs locally on port 4242)
- Chrome or any Chromium browser
